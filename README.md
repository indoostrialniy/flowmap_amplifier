# Flowmap amplifier v1.0.0

### Goals:
If you are experimenting with flowmap technic, sometimes it`s needed to change the whole image velocity vectors encoded as color components.
<br>This program helps do that quickly by just multiplying them on a specified constant.

### Usage:
Drag and drop an image onto the program shortcut and type reqiured amplifying value, the image will be rewrited.

### Warning!
Currently it works only with PNG-images!
